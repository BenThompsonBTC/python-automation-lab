# Python Automation Lab - Codex Instructions

## Repo Purpose

This repo is for practical Python automation that could plausibly help in IT, systems administration, DevOps, reporting, or operational cleanup.

Prefer:

- Small scripts with real utility
- Clear CLI usage
- Tests with pytest
- Type hints where helpful
- Example input/output files using fake data
- GitHub Actions validation

Avoid:

- Giant frameworks unless needed
- Overengineered abstractions
- Scripts that only demonstrate syntax without practical value

Keep code readable and beginner-reviewable.
